# AI 发音翻译

这是一个基于 AI 模型的拼音翻译工具，可以通过输入拼音来搜索对应的语言。

## 用途
很多时候我们在学习一门语言时，只记得单词的发音但是不知道怎么写，这个工具可以帮助我们快速找到对应的单词。
